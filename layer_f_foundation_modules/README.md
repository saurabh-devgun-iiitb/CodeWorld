# Layer F — Foundation Modules (General System Design)

Layer F adds five lightweight but critical modules focused on API-first design and core architecture decisions.

## Modules

- **F1 — API & Request Lifecycle**: `F1_api_request_lifecycle.md`
- **F2 — Data Modeling & Storage Basics**: `F2_data_modeling_storage.md`
- **F3 — Caching & Performance Layer**: `F3_caching_performance.md`
- **F4 — Stateful vs Stateless Systems**: `F4_stateful_vs_stateless.md`
- **F5 — System Evolution & Migration**: `F5_system_evolution_migration.md`

## Outcomes

After completing Layer F, you should be able to confidently handle:

- interview-style system design,
- API-first architecture thinking,
- product-level tradeoff discussions.

Each module markdown is paired with a runnable Python artifact (`F1...py` to `F5...py`) with logging for easy tracing.
